# Jusk
A programing language that compile to c++

### Example for hello world
```rs
@pkg main
@import "Std"
pub fn main()Int{
    Std.Console.Println("Hello World!!")
} 
```
### How to install? (require go,g++ installed)
- ## 1 
```bash
git clone https://github.com/Troxsoft/Jusk.git
```
- ## 2
```bash
go build jusk.go
```
- ## 3 *(optional)*
Add to path
